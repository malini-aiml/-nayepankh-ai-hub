## 🌟 Executive Summary
This project is a mobile-responsive single-page application (SPA) designed exclusively for **NayePankh Foundation**. Built utilizing **HTML5, CSS3, and JavaScript**, the core objective is to leverage artificial intelligence to automate and scale community outreach. By incorporating an interactive simulated NLP layer, the application intelligently handles high-volume user queries from prospective volunteers, students, and donors, boosting the foundation's remote engagement framework.

---

## 🛠️ Core App Architecture & Modules
The application layout mimics a native mobile application environment with a fluid fluid tab navigation bar. It is split into four primary workflows:

* **🏠 Home Screen Dashboard:** Features a modern visual layout highlighting the NGO's slogan ("Spreading Happiness") alongside smooth custom micro-interactions (`transform: scale(0.95)`) engineered for touchscreens.
* **💬 AI Chat Hub:** A clean messaging view that simulates human-to-bot interaction using asynchronous delays. Includes smart prompt chips to streamline frequently asked questions regarding internships, camps, and logistics.
* **🤝 AI Volunteer Matcher:** An interactive data-capture module that acts as an automated profile scanner, matching user skill sets to local or virtual social impact programs.
* **ℹ️ Our Mission (About):** Highlights the deep social impact values of NayePankh, integrated with native look-and-feel notification alerts emphasizing "Tech for Good."

## 🔥 Technical Implementations (Option 2 Requirements Met)
1.  **AI Coding Tool Execution:** Structured and refined leveraging generative AI assistance to optimize standard coding practices.
2.  **Single-Page Multi-Screen Architecture:** Fluid view-switching using dynamic CSS class selectors (`.active { display: flex; }`) ensuring zero page-reload overhead.
3.  **Client-Side Knowledge Router:** Employs an efficient string-parsing keyword router mapped against a structured JSON dataset to simulate intelligent live NLP processing.
